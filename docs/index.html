Methods

Data Processing Pipeline

Raw sequencing reads were processed using FastQC for quality control. Alignment was performed using STAR aligner against the appropriate reference genome (GRCh38/mm10). Read counting was performed using featureCounts.

Differential Expression Analysis

Differential expression analysis was performed using DESeq2 in R. Genes with an adjusted p-value < 0.05 and |log2FoldChange| > 1 were considered significantly differentially expressed.

Data Format

Data is stored in .parquet format for efficient columnar compression and web-based retrieval.

About SIRT6.db

SIRT6.db is a curated repository of transcriptomic data related to the Sirtuin 6 (SIRT6) protein. It aims to facilitate research into aging, metabolism, and genomics.
