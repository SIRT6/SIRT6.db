Methods

Draft placeholder for author review. Differential-expression analyses were performed per organism and per contrast using DESeq2, with median-of-ratios normalized counts, contrast-specific low-count filtering, WT as the reference group, and covariate-adjusted designs where metadata supported them.

Cross-species knockout-response results were harmonized through one-to-one human ortholog mapping and modeled with a three-level phylogenetic random-effects meta-analysis using metafor::rma.mv(), separating phylogenetic, experiment-level, and residual variance components.

Sample quality was assessed during processing with PCA and inter-sample correlation checks. QC plots are not part of this public browser build.

About

SIRT6.db is a public companion resource for transcriptomic studies of SIRT6 perturbation in aging-related biology. It is designed for browsing, visualization, and download of the publication data package.
